server
开发一个支持多用户在线的 FTP 程序：
	1、用户加密认证
	2、允许同时多用户登陆
	3、每个用户都有自己的家目录，且只能访问自己的家目录
	4、对用户进行磁盘配额，每个用户的可用空间不同
	5、允许用户在 ftp server 上随意切换目录
	6、允许用户查看当前目录下文件
	7、允许上传和下载文件，保证文件一致性
	8、文件传输过程中显示进度条
	9、附加功能：支持文件的断点续传

==================================================================================
s1、用户加密认证
    #c 通过 sys.argv 获取所有参数
        保存成json的形式,
    #s 将用户名、密码保存到文件，通过section形式读写

s2、允许同时多用户登陆
    通过serversocket完成多用户

s3、每个用户都有自己的家目录，且只能访问自己的家目录
    登录进去后创建一个自己的 home 目录
    cd 的时候需要校验是否已经到根目录

s4、对用户进行磁盘配额，每个用户的可用空间不同
    上传文件之前需要判断空间是否足够
    在保存文件的时候，计算已经使用的空间大小，并保存到用户密码的文件中

s5、允许用户在 ftp server 上随意切换目录
    可以执行 cd..  cd dir 操作

s6、允许用户查看当前目录下文件
    可以执行ls

sc7、允许上传和下载文件，保证文件一致性
    通过md5 进行校验

c8、文件传输过程中显示进度条
    sys模块

sc9、附加功能：支持文件的断点续传
    通过文件的seek进行设置

sc10、记录操作和异常到日志文件

# 数据传输时的通讯格式
客户端->服务端：{"cmd": cmd, }
服务端->客户端：{"status": ..}
